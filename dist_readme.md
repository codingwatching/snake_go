# 🎮 贪吃蛇游戏：快速开始指南 (Quick Start Guide)

欢迎体验 Snake Go **v3.1.0**！本指南旨在帮助非技术用户快速启动游戏并了解全新玩法。

## 📥 解压说明
解压下载到的 `.zip` 压缩包，你将看到适用于各操作系统的运行程序。

---

## 🚀 方式 A：网页版 (推荐)
**网页版支持精美画面、多维度音效、火球对战以及手机全触控操作。**

### 1. 启动服务器 (根据系统选择)
在 `dist` 文件夹中找到对应系统的程序并运行：
- **Windows**: 双击 `webserver_windows.exe`
- **Mac (M1/M2/M3)**: 运行 `webserver_mac_arm64`
- **Mac (Intel)**: 运行 `webserver_mac_amd64`
- **Linux**: 运行 `webserver_linux`

### 2. 进入游戏
- **本电脑玩**: 打开浏览器，输入 👉 `http://localhost:8080`
- **📱 手机或其他电脑玩**: 
  1. 确保手机与电脑连接在**同一个 Wi-Fi**。
  2. 找到你电脑的 **局域网 IP 地址**：
     - **Windows**: 在终端输入 `ipconfig`，找 "IPv4 地址"（通常是 `192.168.x.x`）。
     - **Mac**: 按住 `Option` 键点击顶部 Wi-Fi 图标，即可看到 IP 地址。
  3. 在手机浏览器输入：`http://你的IP地址:8080` (例如 `http://192.168.1.5:8080`)

## ⚔️ 全新玩法说明 (v3.1.0 特有)

### 1. 游戏模式 🧘⚔️
- **🧘 禅模式 (Zen)**: 休闲练习。没有时间限制，没有 AI 干扰，适合刷高分或放松。
- **⚔️ 竞技模式 (Battle)**: 30秒限时决斗。你将与一个**智能 AI** 同台竞技，看谁在时间结束前分数更高！

### 2. 火球对战系统 🔥
- **发射火球**: 按 **F** 键或点击手机上的火球图标。
- **爆头奖励 (Headshot)**: 击中 AI 蛇的头部可将其**眩晕 2 秒**并获得 50 分！
- **攻击到蛇身其他部位**: 击中 AI 蛇的其他部位，蛇的长度-1， 同时获得20分。
- **摧毁障碍**: 火球可以击碎随机出现的岩石障碍，获得额外分数。

### 3. 位置奖励机制 �
- **墙角奖励**: 在地图最角落吃到豆子可获得 **+100 分**！
- **边缘奖励**: 在墙边吃到豆子获得 **+30 分**。
- **技巧挑战**: 离墙越近，风险越高，收益越大。

## �🎮 操作说明

### 💻 键盘操作：
- **方向键 / W A S D**: 控制转向。
- **长按方向键**: 触发 **🚀 加速 (Boost)**。
- **F 键**: 发射 **🔥 火球** (仅限网页版)。
- **A 键**: 开启/关闭 **🤖 AI 自动驾驶** (仅限网页版)。
- **P / 空格**: 暂停/继续。
- **R**: 死亡后重新开始。

### 📱 手机操作：
- **虚拟摇杆/按键**: 控制转向。
- **长按转向键**: 触发加速。
- **🔥 图标**: 发射火球。
- **🤖 图标**: 切换自动驾驶。

## 💡 常见问题
- **画面遮挡？** 我们优化了奖励消息的大小和位置，并在 v3.1.0 中大幅缩短了停留时间。
- **怎么重新开始？** 游戏结束后直接点击屏幕中间或按 **R** 键。
- **没声音？** 网页版需在进入后点击一次页面方可激活合成音效。
